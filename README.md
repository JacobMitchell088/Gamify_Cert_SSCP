# Gamify_Cert_SSCP

A gamified study site for the (ISC)² **Systems Security Certified Practitioner (SSCP)** certification. Anonymous play — no login required. Pick a mini-game from the menu and play 30 SSCP-domain practice questions generated by free LLMs via OpenRouter.

See [GAMES.md](./GAMES.md) for the full catalogue of mini-game ideas.

## Status (current)

- ✅ Backend: FastAPI, SQLite question pool, seed-on-startup, `/run/*` endpoints.
- ✅ OpenRouter racer + APScheduler background refill (no-ops without an API key — fine for local play).
- ✅ Frontend: Vite + React + TypeScript + Tailwind + Zustand + Phaser 3.
- ✅ Mini-game: **Tower Defense: Exploit Wave** — place towers by answering, defend the core for 30 questions.
- ✅ Mini-game: **Hacker Boss Duel** — RPG combat against the Mecha-stone Golem. Correct answers parry-counter, wrong answers take a hit; collect relic modifiers from treasure chambers.
- ✅ Mini-game: **Vault Lockdown** — six threat paths close in on a central vault. Correct = drop a lock on a path of your choice; wrong = attackers gain an extra step. Don't let any path reach the vault.
- ✅ Run summary + missed-question review with per-domain breakdown.

**Active games:** Tower Defense, Hacker Boss Duel, and Vault Lockdown. (Asteroid Answer Run remains in the codebase as an earlier prototype but is hidden from the menu.)

A run is 30 questions. The selected mini-game persists for the whole run — there is no rotation. The player picks the game on the menu; the backend respects that choice.

## Prerequisites

- **Conda** (for the backend env)
- **Node 18+** (Node 20/22/24 all fine)

## Backend setup

```bash
cd backend
conda env create -f environment.yml
conda activate sscp-backend

# Optional: enable AI question generation
cp .env.example .env
# then edit .env and set OPENROUTER_API_KEY (free key from openrouter.ai)

# Run the API
uvicorn app.main:app --reload
```

The API serves on `http://localhost:8000`. Visit `/docs` for the OpenAPI UI.

On first start the seed JSON is loaded into a local SQLite at `backend/app/data/pool.db`. If `OPENROUTER_API_KEY` is set, a background job tops up the pool with AI-generated questions every 30 s, capped by `MONTHLY_CALL_CAP`.

### One-time bulk seed generation (optional)

After configuring an OpenRouter key, pre-fill the seed JSON with a larger batch:

```bash
cd backend
python -m scripts.generate_seed --per-domain 50
```

Then **spot-check** the additions in `app/data/seed_questions.json` before committing — LLMs hallucinate cert specifics.

## Frontend setup

```bash
cd frontend
npm install
npm run dev
```

Open `http://localhost:5173`, pick a game from the menu, hit **Play**.

### Controls

| Game | Controls |
|------|----------|
| Tower Defense | Click an answer card to place a tower; wave auto-runs. |
| Hacker Boss Duel | Click an answer card; click relic cards in treasure chambers. `SPACE` advances past the feedback panel. |
| Vault Lockdown | Click an answer card; if correct, click any radial path to drop a lock there. `SPACE` advances past the feedback panel. |

## Environment variables

| Var                       | Where    | Purpose                                      |
|---------------------------|----------|----------------------------------------------|
| `OPENROUTER_API_KEY`      | Backend  | Auth for OpenRouter calls.                   |
| `OPENROUTER_FREE_MODELS`  | Backend  | Comma-separated list of `:free` models.      |
| `POOL_REFILL_THRESHOLD`   | Backend  | Trigger refill below this many unused.       |
| `MONTHLY_CALL_CAP`        | Backend  | Hard ceiling on outbound LLM calls/month.    |
| `VITE_API_BASE_URL`       | Frontend | Backend origin (defaults to localhost:8000). |

## Project layout

```
backend/
  app/
    main.py                # FastAPI entry
    config.py              # env/settings
    db.py, models.py       # SQLModel + SQLite pool
    routers/               # /run/*, /pool/stats, /health
    services/
      openrouter.py        # async free-model racer
      pool.py              # question pool + refill
      validator.py         # pydantic question schema
      games.py             # BATCH_SIZE, TOTAL_BATCHES, GAME_ROTATION
    data/
      seed_questions.json
  scripts/
    generate_seed.py

frontend/
  public/
    sprites/               # RPG sprites + EnviroPack02 backdrop
  src/
    api/                   # fetch wrapper
    store/runStore.ts      # Zustand store (run lifecycle, queue, score)
    components/            # Menu, HUD, RunSummary, RunReview
    games/
      GameHost.tsx         # React-Phaser bridge
      TowerDefenseScene.ts # tower defense
      RpgBossScene.ts      # hacker boss duel
```

## Sprite assets

The RPG game uses two free pixel-art packs (zipped at repo root, extracted into `frontend/public/sprites/`):

- **Red Hood Free** (12-frame idle + 12-frame slash) — player character
- **Mecha-stone Golem** (8-frame idle + 8-frame arm-extension shoot) — boss
- **EnviroPack02** (sky, moon, stars) — backdrop

Originals are licensed for free use per their respective itch.io packs.
